# 1-3. Google AI Studio Build로 증강기 만들기

## Google AI Studio란?

[Google AI Studio](https://aistudio.google.com)는 Google이 제공하는 **Gemini API 개발 환경**입니다.
프롬프트를 테스트하고, 모델을 비교하고, API 키를 발급받을 수 있는 개발자 도구입니다.

### AI Studio Build란?

AI Studio 안의 <strong>Build 모드</strong>는 프롬프트만으로 <strong>풀스택 웹앱</strong>을 만들 수 있는 기능입니다.
Gemini Canvas와 비슷하지만, 더 개발자 친화적인 기능을 제공합니다.

| 비교 | Gemini Canvas | AI Studio Build |
|------|--------------|-----------------|
| 접근 방식 | 대화형, 간편한 앱 생성 | 풀스택 앱 개발 환경 |
| 코드 편집 | 단일 HTML 코드 보기/수정 가능 | Code 탭에서 멀티 파일 직접 수정 가능 |
| 백엔드 | 프론트엔드만 | Node.js 런타임 + npm 패키지 지원 |
| 배포 | 공유 링크 | Google Cloud Run 배포, GitHub 연동, ZIP 다운로드 |
| 협업 | 링크 공유 | Annotation 모드로 UI 요소 직접 지정하며 수정 요청 |

<Callout type="info" emoji="📘">
**공식 문서:** [AI Studio Build 모드 문서](https://ai.google.dev/gemini-api/docs/aistudio-build-mode) · [AI Studio 퀵스타트](https://ai.google.dev/gemini-api/docs/ai-studio-quickstart)
</Callout>

## 1-3a. Build 갤러리 탐색 + 수정해보기

<Steps>
### AI Studio 접속하기

[aistudio.google.com](https://aistudio.google.com)에 접속합니다.

### 갤러리 탐색하기

왼쪽 메뉴에서 **Build**를 선택하고, 예시 프로젝트 갤러리에서 마음에 드는 앱 하나를 클릭해보세요.

### 수정해보기

선택한 앱에서 1~2가지를 변경해보세요.

<Callout emoji="✏️">
**변경 예시:**
- "색상을 파란 계열로 바꿔줘"
- "입력 필드를 하나 더 추가해줘"
- "결과를 테이블 형태로 보여줘"
</Callout>

</Steps>

## 1-3b. AI Studio에서 프롬프트 증강기 v2 만들기

강사가 AI Studio Build로 만든 프롬프트 증강기 예시를 먼저 확인해보세요:
[gemini-prompt-refiner](https://gemini-prompt-refiner-487731245387.us-west1.run.app/)

<Steps>
### 새 프로젝트 시작하기

Build에서 **새 프로젝트**를 시작합니다.

### 프롬프트 입력하기

아래 프롬프트를 입력하세요. **최소 1가지는 본인이 수정**해서 입력해보세요.

```
프롬프트 증강기 웹앱을 만들어줘.

기능:
- 사용자가 프롬프트를 입력하면 3가지 버전으로 개선해서 보여줘
  - 짧은 버전: 핵심만 간결하게
  - 중간 버전: 맥락과 제약조건 포함
  - 상세 버전: 역할, 맥락, 출력 형식, 제약조건 모두 포함
- 각 버전마다 복사 버튼
- Google의 프롬프트 엔지니어링 모범 사례를 기준으로 개선

UI:
- 상단에 입력창, 하단에 3개 탭으로 결과 표시
- 모바일에서도 잘 보이게
```

### 테스트하기

생성된 앱을 테스트해보세요.

### Gemini Canvas 버전과 비교하기

- 어떤 점이 다른가요?
- 어떤 도구가 더 편했나요?
</Steps>
