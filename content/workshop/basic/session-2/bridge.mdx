# 2-2. 반복을 발견한다

## 내가 AI에게 반복한 말 찾기

세 도구로 같은 앱을 만들면서 AI에게 반복했던 말을 적어보세요.

### 떠올려볼 질문

- 세 도구 모두에서 비슷하게 요청한 것은?
- 첫 결과물을 보고 매번 고친 것은?
- "당연히 이렇게 해야 하는데 왜 안 했지?" 싶었던 것은?
- 가이드 프롬프트에서 **내가 직접 수정하거나 추가한 부분**은? — 이것도 반복입니다

### 예시

| 내가 반복한 말 | 카테고리 |
|---------------|---------|
| "모바일에서도 잘 보이게" | UI 규칙 |
| "노션처럼 깔끔하게" | 디자인 스타일 |
| "로딩 중이라고 표시해줘" | UX 패턴 |
| "버튼 누르면 바로 결과 보여줘" | 인터랙션 |
| "한국어로 해줘" | 언어 |
| (가이드에 없었는데 내가 추가로 요청한 것) | 내 패턴 |
