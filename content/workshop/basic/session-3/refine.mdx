# 3-3. 내가 만든 증강기로 AGENTS.md 다듬기

세션 1~2에서 만든 프롬프트 증강기를 활용해 AGENTS.md를 다듬어봅니다.

## Step 1: 증강기로 프롬프트 만들기

<Steps>
### 증강기 열기

본인이 만든 증강기를 열어주세요.

### 프롬프트 입력하기

아래 프롬프트를 증강기에 입력하세요.

```
내 AGENTS.md 초안을 리뷰하고 개선해줘. 뻔한 내용은 빼고, 구체적인 규칙만 남겨줘.
```

### 결과 복사하기

증강기가 만들어준 **개선된 프롬프트**를 복사하세요.
</Steps>

---

## Step 2: 개선된 프롬프트로 AGENTS.md 다듬기

<Steps>
### Lovable 채팅에 입력하기

Lovable 채팅에 증강기가 만든 프롬프트를 붙여넣고, 그 아래에 본인의 AGENTS.md 초안을 추가하세요.

```
(증강기가 만들어준 개선된 프롬프트 붙여넣기)

---
(본인의 AGENTS.md 초안 붙여넣기)
```

### 결과 확인하기

AI가 개선해준 버전을 확인합니다.

### AGENTS.md 업데이트하기

마음에 드는 부분을 골라 AGENTS.md를 업데이트하세요.

### Knowledge에 반영하기

업데이트한 AGENTS.md를 Knowledge에도 반영하세요.
</Steps>

<Callout type="info" emoji="🔄">
**자기 참조 루프:**

내가 만든 증강기로 프롬프트를 다듬고
→ 그 프롬프트로 AGENTS.md를 개선하고
→ 더 나은 AGENTS.md로 다음에 더 잘 만든다
</Callout>
