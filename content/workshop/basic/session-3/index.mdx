# 세션 3: 반복을 자산으로 만든다

**목표**: 반복을 AGENTS.md로 정리하고, Before/After로 차이를 직접 확인한다

## AGENTS.md란?

브릿지에서 적은 "반복한 말"을 기억하시나요?
AGENTS.md는 그 반복을 파일 하나에 담아서, AI에게 "나는 이렇게 일한다"를 알려주는 것입니다.
한 번 작성하면 매번 반복해서 말하지 않아도 AI가 알아서 적용합니다.

<Callout type="info" emoji="💡">
**한 줄 요약**

AGENTS.md = "반복한 말"을 한 번만 적어두면, AI가 매번 알아서 적용하는 파일
</Callout>
