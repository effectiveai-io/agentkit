## 두 워크숍의 차이

| | 워크숍 기본 | 워크숍 심화 |
|---|---|---|
| **핵심 질문** | "AI에게 매번 하는 말이 뭔가?" | "그 말을 체계화하면 얼마나 나아지는가?" |
| **접근** | 세 도구로 같은 앱을 만들며 반복 발견 | 하나의 도구로 지침서를 3사이클 개선 |
| **결과물** | AGENTS.md 초안 | 평가 기준 + 검증된 업무 지침서 v2 |
| **영감** | Karpathy의 에이전틱 엔지니어링 선언 | Karpathy의 autoresearch 프로젝트 |
| **사례** | 업계 사례 (컬리, 당근, 우아한형제들) | 우아한형제들 기술 블로그 심층 연결 |
