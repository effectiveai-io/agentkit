# 미션 3: AI API 연동

**시간**: 25분

## 목표

Google AI Studio에서 발급한 Gemini API 키를 연결해, 버튼을 누르면 실제 AI가 프롬프트를 개선해주도록 만든다.

## 실습

<Steps>
### API 키 환경변수 설정

Lovable 프로젝트 → `Settings` → `Environment Variables` → 추가:

```
VITE_GEMINI_API_KEY=여기에_API_키_붙여넣기
```

<Callout type="warning">
API 키는 절대 코드에 직접 넣지 마세요. 환경변수로 관리해야 GitHub에 노출되지 않습니다.
</Callout>

### System Prompt 커스터마이징 (선택)

AI Studio에서 실험했던 System Prompt 중 마음에 드는 것을 넣어보세요.

더 좋은 결과를 내는 System Prompt를 찾아보는 것도 좋습니다.

### 최종 테스트

1. 프롬프트 입력 (예: "ChatGPT에게 여행 계획 짜달라고 하기")
2. '개선하기' 클릭
3. AI가 개선한 프롬프트 확인
4. 저장하기 → 목록에서 확인
</Steps>

## 완성된 앱 체크리스트

- [ ] Google 로그인/로그아웃
- [ ] 프롬프트 입력 → AI 개선 → 결과 표시
- [ ] 저장하기 → 내 프롬프트 목록
- [ ] Publish URL로 팀에 공유 가능
