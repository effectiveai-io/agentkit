# 세션 2: 에이전틱 고도화

**시간**: 60분 | **목표**: 바이브 코딩으로 만든 앱을 에이전틱 관점으로 고도화한다

## 바이브 코딩을 넘어서

세션 1에서 만든 프롬프트 증강기는 UI만 있는 앱입니다.
이것을 **실제로 동작하는 서비스**로 만들려면 도구를 연결하고 구조를 설계해야 합니다.

```
세션 1 결과물 (UI만 있는 앱) ← 바이브 코딩 영역
    ↓
미션 1: Google 로그인 추가 (인증 설계)
    ↓
미션 2: 프롬프트 저장/불러오기 (데이터 구조 설계)
    ↓
미션 3: Gemini API 연결 (AI 에이전트 연동) ← 에이전틱 코딩 영역
```

<Callout type="warning">
**미션은 순서대로** 진행하세요. 각 미션은 이전 미션의 결과물 위에 쌓입니다.
막히면 강사에게 질문하거나 힌트를 확인하세요. 완성이 목적이 아니라 **경험이 목적**입니다.
</Callout>
