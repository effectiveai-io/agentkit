# 세션 2: Lovable 고도화

**시간**: 60분 | **목표**: 세션 1에서 만든 앱을 실제로 배포 가능한 수준으로 고도화한다

## 오늘 추가하는 것

세션 1에서 만든 프롬프트 증강기에 세 가지를 추가합니다.

```
세션 1 결과물 (UI만 있는 앱)
    ↓
미션 1: Google 로그인 추가
    ↓
미션 2: 프롬프트 저장/불러오기 (Supabase DB)
    ↓
미션 3: Gemini API 연결 (실제로 동작하는 AI)
```

<Callout type="warning">
**미션은 순서대로** 진행하세요. 각 미션은 이전 미션의 결과물 위에 쌓입니다.
막히면 강사에게 질문하거나 힌트를 확인하세요. 완성이 목적이 아니라 **경험이 목적**입니다.
</Callout>
