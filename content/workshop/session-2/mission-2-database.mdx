### Supabase 테이블 확인

Lovable이 자동으로 테이블을 만들었는지 확인합니다.

Supabase 대시보드 → `Table Editor` → `prompts` 테이블 확인

<Callout type="info">
**RLS (Row Level Security)**란?

"내 데이터는 내가 로그인했을 때만 볼 수 있다"는 보안 규칙입니다.
Lovable이 자동으로 설정해주지만, 어떤 개념인지 알아두면 좋습니다.
</Callout>

### 테스트

1. 프롬프트 입력 → 개선 (아직 AI 미연동이라 mock 결과)
2. '저장하기' 클릭
3. 목록에서 저장된 항목 확인
4. 다른 브라우저/시크릿 탭에서 로그인 → 내 프롬프트만 보이는지 확인
</Steps>

## 확인 포인트

- [ ] 저장 후 목록에 나타난다
- [ ] 로그아웃 상태에서는 저장 목록이 보이지 않는다
- [ ] Supabase 대시보드에서 실제 데이터를 확인할 수 있다
