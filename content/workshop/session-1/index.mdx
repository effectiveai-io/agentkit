# 세션 1: 프롬프트 증강기 만들기

**시간**: 60분 | **목표**: 같은 앱을 3가지 도구로 만들며 도구의 차이를 직접 체감한다

## 오늘 만드는 것

**프롬프트 증강기** — 내가 쓴 프롬프트를 더 좋은 프롬프트로 다듬어주는 앱

> [달각(dalgak.com)](https://dalgak.com) 처럼, 프롬프트를 입력하면 AI가 더 명확하고 구체적인 프롬프트로 개선해줍니다.

### 앱의 핵심 기능

```
사용자가 프롬프트 입력
    ↓
AI가 부족한 부분을 질문하거나 직접 개선
    ↓
개선된 프롬프트 출력 + 원본과 비교
```

<Callout type="warning">
**왜 이걸 세 번 만드나요?**

같은 목표라도 도구에 따라 걸리는 시간, 결과물의 완성도, 이후 확장 가능성이 완전히 다릅니다.
세 번 만들어보면 "상황에 맞는 도구 선택 기준"이 생깁니다.
</Callout>
