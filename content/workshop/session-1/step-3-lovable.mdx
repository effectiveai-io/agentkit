## Step 1~3 비교

| | Gemini Canvas | Google AI Studio | Lovable |
|---|---|---|---|
| **시작 속도** | 즉시 | 즉시 | 1분 |
| **UI 품질** | 기본 | 없음(API만) | ⭐⭐⭐ |
| **배포** | ❌ | ❌ | ✅ URL 생성 |
| **코드 소유** | ❌ | ❌ | ✅ GitHub |
| **확장성** | 낮음 | 중간(API) | 높음 |
| **이후 수정** | 대화로만 | 프롬프트로만 | 코드 직접 편집 가능 |

<Callout type="info">
**Lovable의 진짜 강점**은 "첫 버전 이후"입니다.
만들어진 코드를 로컬에서 수정하거나, GitHub으로 팀과 협업하거나, 세션 2에서처럼 고도화할 수 있습니다.
