# vibekit

> 좋은 Input이 좋은 Output을 만든다.
>
> 단순한 도구도 제대로 쓰면 더 강력해진다.

vibekit은 **AI 도구 활용 실습 가이드**입니다.
프롬프트를 잘 쓰는 것부터 시작해, 실제로 배포 가능한 앱을 만드는 것까지 — 단계별로 경험하며 배웁니다.

## 이 가이드에서 배우는 것

- **프롬프트 자산화**: 내가 쓴 프롬프트를 버리지 않고 팀의 자산으로 만드는 법
- **같은 목표, 다른 도구**: Gemini Canvas → Google AI Studio → Lovable로 같은 앱을 세 번 만들며 도구의 차이를 체감
- **바이브 코딩 고도화**: Google 인증, 데이터베이스, AI API 연동까지 단계별로

## 핵심 관점

```
프롬프트 작성 → 증강 → 저장 → 공유 → 팀 자산
```

달각(dalgak.com)처럼 프롬프트를 한 번 잘 만들어두면 반복해서 쓸 수 있습니다.
이 워크숍에서는 그 도구를 **직접 만들어**봅니다.
